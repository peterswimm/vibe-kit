# Ignite 2025 Highlights (for Event Agents)

## Agents & Governance
- Agent 365 + Entra Agent ID as the unified identity/policy plane for agent fleets.
- Foundry Agent Service + Control Plane for hosted agents, multi-agent graphs, and observability.
- Copilot Studio evaluations and agent flows integrated with Agent 365 policies.

## Productivity & Work IQ
- Work IQ as cross-surface memory and relevance engine for M365 + Copilot experiences.
- Copilot Pages/Notebooks improvements for chaining tools and sharing explainable reasoning.
- Outlook/Teams/Word/Excel/PPT agents updated to honor Work IQ and Agent 365 policies.

## Data & MCP
- Fabric IQ and Foundry IQ for semantic knowledge + RAG with governance hooks.
- Power Apps MCP, Dataverse MCP, Dynamics MCP make enterprise connectors MCP-native.
- SharePoint MCP authoring APIs (pages/lists) with Purview labeling support.

## Edge & Windows
- Edge for Business Copilot Mode with multitab reasoning and agent mode extensibility.
- Windows agent workspace + Windows MCP connectors for local files and system context.

## Security & Compliance
- Purview and Defender expanded to agent actions and MCP tool calls.
- DLP-aware tool invocation patterns; audit trails routed to Agent 365/Foundry Control Plane.
- Conditional Access for agents (per-tool, per-surface) using Entra Agent ID.
