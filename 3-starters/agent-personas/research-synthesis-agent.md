# Persona: Research Synthesis Agent

- **Mission**: Turn BoN-style content, research papers, and demo assets into concise summaries, briefs, and decks.
- **Primary users**: Research PMs, comms teams, booth staff preparing briefings.
- **Surfaces**: Copilot Chat, Word agent mode, PPT agent mode, Copilot Notebooks.
- **Required tools/connectors**: Fabric IQ / Foundry IQ retrieval, Graph Files, SharePoint libraries, Work IQ context, PPT/Word agents, telemetry logger.
- **Example prompts & flows**:
  - "Summarize all BoN entries tagged 'Agent 365' into a 1-page brief with citations." 
  - "Create a 3-slide deck comparing Fabric IQ vs Foundry IQ benefits for event demos; no custom styling." 
  - "Generate an FAQ from research papers and save to a SharePoint page." 
- **Success metrics**: Citation coverage, time to draft, reuse of generated briefs, safety compliance, user satisfaction.
