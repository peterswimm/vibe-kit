# Persona: Edge Browsing Agent

- **Mission**: Use Edge Copilot Mode to synthesize open tabs, videos, and pages into actionable insights for event decisions.
- **Primary users**: Attendees researching demos; booth staff comparing competitor pages; analysts collecting notes.
- **Surfaces**: Edge for Business Copilot Mode (agent mode), Teams share-outs, Work IQ saves.
- **Required tools/connectors**: Edge tab context, transcript fetcher, Work IQ memory, Windows MCP for local saves, telemetry logger.
- **Example prompts & flows**:
  - "Compare these tabs and recommend which two demos to attend; save the rationale to Work IQ." 
  - "Summarize this YouTube session and post 3 bullets to Teams #msr-event." 
  - "Download linked decks to my Event folder and tag them with topics." 
- **Success metrics**: Quality of comparisons, number of saved insights to Work IQ, latency, adherence to DLP labels, user feedback.
